# EduFlow ERP 🎓

EduFlow is a comprehensive, high-performance School Management System (ERP) built to modernize administrative, academic, and financial operations for educational institutions. 

Built with the **TALL Stack** (Tailwind, Alpine.js, Laravel, Livewire), this system eliminates page reloads, providing a seamless, "desktop-app" experience in the browser.

## 🚀 Core Modules

### 1. Financial Desktop (Point-of-Sale Style)
* **Automated Fee Generation:** Generate monthly fee vouchers for the entire student body with a single click.
* **Instant Defaulter Tracking:** Live tracking of pending payments and defaulting students.
* **Cash Collection & Receipts:** One-click cash collection with instantly printable, professional receipts.
* **Ad-Hoc Billing:** Apply custom surcharges (e.g., library fines, ID card replacements) directly to a student's ledger.

### 2. Academics & Grading
* **Curriculum Planner:** Define class structures, assign subjects, and set standard grading parameters.
* **Dynamic Gradebook:** Frictionless interface for teachers to input subject marks and toggle absentee statuses.
* **Mass Report Card Dispatch:** Instantly generate and print structured report cards by class and exam session.

### 3. Identity & HR Management
* **Role-Based Access Control (RBAC):** Strict security protocols ensuring faculty, finance officers, and admins only access their respective modules.
* **Digital Staff Dossiers:** Comprehensive digital records for faculty, including base salaries and joining dates.
* **Registry Onboarding:** Streamlined registration and admission processes for both students and staff.

## 💻 Tech Stack
* **Framework:** Laravel 11
* **Frontend:** Livewire 3 (Volt) & Alpine.js
* **Styling:** Tailwind CSS
* **Database:** MySQL / Eloquent ORM
* **Architecture:** SPA-like reactive interfaces using DOM morphing.
